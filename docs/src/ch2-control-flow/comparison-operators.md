# Comparison operators

Valid comparison operators are as follows:

| Operator | Meaning               |
| ---------|-----------------------|
| &&       | And                   |
| \|\|     | Or                    |
| ==       | Equal                 |
| !=       | Not equal             |
| >        | Greater than          |
| <        | Less than             |
| >=       | Greater than or equal |
| <=       | Less than or equal    |

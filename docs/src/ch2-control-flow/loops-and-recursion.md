# Loops & Recursion

## For Statements

### For/In Statements
```rs,ignore
let array = ["Item 1", "Item 2", "Item 3"];
for item in array {
    println(item)
}
```
See [arrays](../ch1-variables/arrays.md) for more information.

### For/Iterator Statements
```rs,ignore
for i in 1..10 {
    println(i);
}
```

## While Statements
```rs,ignore
let i = 0;
while i < 10 {
    println(i);
    i++;
}
```
